### Description

The Free Image CDN plugin enhances your website's performance and speeds up image loading using the free CDN service powered by [wsrv.nl](https://wsrv.nl/). This plugin is designed to simplify the integration of the wsrv.nl image CDN into your WordPress site, making it easier to deliver your images quickly and efficiently to your visitors.

### Key Features

- **Crazy Fast:** Performance is not just an afterthought; it's a core feature. Enjoy lightning-fast loading times for your images.
- **Awesome Technology:** We employ best-in-class technologies such as nginx and libvips to ensure top-notch image processing.
- **Free / Open Source:** wsrv.nl is an open-source project, so you can save your money for other essential tasks.
- **Worldwide Cache:** Benefit from a global cache that manipulates images on-the-fly, ensuring optimal performance.
- **CDN Caching:** Leverage Cloudflare's global network of 200+ data centers to cache and deliver your images for the fastest load times.
- **Widely Used:** Join the community of users who process 6 million images per hour, generating around 400TB of outbound traffic each month.
- **Simple Install:** Easily integrate the CDN service into your WordPress website for quick and hassle-free image processing.

== Installation ==

1. Upload the `free-image-cdn` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= What is a CDN (Content Delivery Network)? =

A CDN is a network of servers distributed across various locations globally, designed to deliver web content, including images, more efficiently to users. By using a CDN, you can reduce the load on your web hosting server and improve the speed and reliability of content delivery.

= Why should I use a CDN for images? =

Using a CDN for images offers several advantages, including faster loading times, reduced server load, improved performance, and a better user experience. CDNs store cached copies of your images on multiple servers worldwide, ensuring that users can access your content quickly, no matter where they are located.

= What is [wsrv.nl](https://wsrv.nl/)? =

[wsrv.nl](https://wsrv.nl/) is a free and reliable image CDN service that provides a simple way to speed up image loading on your website. It supports a wide range of image formats and can be used to deliver images from a CDN without any additional cost.
